# Executive Function Skills Questionnaire

Based on the Dawson & Guare Executive Skills framework from *Smart but Scattered*.

---

## Overview

Executive functions are the cognitive skills that help us manage ourselves and our resources to achieve goals. This assessment measures 12 executive function skills, particularly relevant for people with ADHD.

---

## Instructions

**Step 1:** Read each item below and rate how well it describes you.

**Step 2:** Use the rating scale to choose the appropriate score for each statement.

**Step 3:** Calculate the total score for each skill section (3 questions per skill).

**Step 4:** Use the interpretation key to identify your strengths and challenge areas.

## Interpretation Key

### Score Ranges for Individual Skills

| Score Range | Interpretation |
|-------------|----------------|
| 18-21 | **Strong** — Natural strength, leverage this skill |
| 12-17 | **Moderate** — Functional, can improve with practice |
| 6-11 | **Challenge** — Needs scaffolding and support systems |
| 3-5 | **Significant Challenge** — Requires substantial external support |

### Identifying Patterns

**Top 3 Scores = Strongest Skills**
- These are your natural strengths to leverage
- Build systems that play to these abilities

**Bottom 3 Scores = Challenge Areas**
- These need external scaffolding
- Don't rely on willpower alone—create systems

---

## ADHD-Specific Insights

Common ADHD challenge patterns:
- **Working Memory** + **Sustained Attention** = Classic attention challenges
- **Task Initiation** + **Time Management** = Procrastination pattern
- **Emotional Control** + **Response Inhibition** = Impulsivity pattern
- **Organization** + **Planning/Prioritization** = Executive organization challenges

---

## Retake Frequency

**Quarterly retakes recommended** to:
- Track improvement over time
- Identify which scaffolding strategies are working
- Adjust systems based on changing demands
